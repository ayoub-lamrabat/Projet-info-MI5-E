PROJET CY-MEMO-RPG — GUIDE D'UTILISATION
======================================================

AUTEURS : [Vos Noms de Trinôme]
FILIÈRE : préING1 (2025-2026)

--- COMMENT COMPILER LE PROJET ---
Ouvrez votre terminal dans le dossier et tapez :
$ make

Cela génère automatiquement l'exécutable nommé "memo_rpg".

--- COMMENT LANCER LE JEU ---
Tapez la commande suivante dans votre console :
$ ./memo_rpg

--- OPTIONS DE NETTOYAGE ---
Pour supprimer les fichiers objets compilés intermédiaires (.o), tapez :
$ make clean
